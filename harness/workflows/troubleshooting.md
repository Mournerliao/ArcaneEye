# ArcaneEye 故障排除指南

## 常见问题及解决方案

### 1. 截图功能问题

#### 问题：截图功能无法使用
**症状**：按下快捷键后无法截图
**解决方案**：
1. 检查应用是否有屏幕录制权限
2. 检查系统权限设置
3. 重启应用

#### 问题：HDR 截图发白
**症状**：在 HDR 显示器上截图显示为白色
**解决方案**：
1. 确认显示器是否开启 HDR
2. 尝试使用内置截图功能
3. 可选：安装第三方 HDR 截图工具（如 Lumiere）获得更好体验
4. 更新显卡驱动程序
5. 检查 Windows HDR 设置

### 2. Tauri 应用问题

#### 问题：Tauri 应用无法启动
**症状**：`bun run tauri dev` 失败
**解决方案**：
1. 检查 Rust 工具链是否正确安装
2. 检查 Tauri CLI 版本
3. 清除构建缓存：`cargo clean`
4. 重新安装依赖：`bun install`

#### 问题：前端热重载不工作
**症状**：修改代码后页面不自动刷新
**解决方案**：
1. 检查 Bun 版本
2. 清除 node_modules：`rm -rf node_modules && bun install`
3. 检查端口是否被占用
4. 重启开发服务器

### 3. 数据同步问题

#### 问题：ARAMGG 数据同步失败
**症状**：无法获取最新胜率数据
**解决方案**：
1. 检查 API Key 是否正确
2. 检查网络连接
3. 检查 API 服务状态
4. 查看同步日志

#### 问题：数据库连接失败
**症状**：无法连接 Supabase 数据库
**解决方案**：
1. 检查 Supabase 配置
2. 检查网络连接
3. 检查数据库状态
4. 重置数据库连接

### 4. AI 集成问题

#### 问题：英雄识别失败
**症状**：无法识别截图中的英雄
**解决方案**：
1. 检查截图质量
2. 检查 AI 模型 API Key
3. 优化 Prompt 设计
4. 尝试不同的 AI 模型

#### 问题：推荐结果不准确
**症状**：推荐的海克斯或出装不符合预期
**解决方案**：
1. 检查数据源准确性
2. 优化 RAG 数据增强
3. 调整 Prompt 参数
4. 收集用户反馈优化

### 5. 性能问题

#### 问题：截图处理缓慢
**症状**：截图后长时间无响应
**解决方案**：
1. 优化图片处理算法
2. 使用缓存减少重复处理
3. 异步处理避免阻塞
4. 监控系统资源使用

#### 问题：内存使用过高
**症状**：应用占用内存过多
**解决方案**：
1. 检查内存泄漏
2. 优化数据结构
3. 及时释放资源
4. 使用内存分析工具
